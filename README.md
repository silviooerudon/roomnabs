# roomnabs

A [Next.js](https://nextjs.org) (App Router) site, configured for deployment on
[Netlify](https://www.netlify.com).

## Stack

- Next.js 16 (App Router) + React 19
- TypeScript
- Deployed on Netlify via the official OpenNext-based Next.js adapter

## Local development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm start        # run the production build locally
```

## Routes

- `/` — home
- `/go/[linkId]` — dynamic, server-rendered redirect endpoint (e.g. `/go/github`)
- `/sitemap.xml` — generated by `app/sitemap.ts`
- `/robots.txt` — generated by `app/robots.ts`

## Netlify deployment

Netlify auto-detects Next.js and provisions its Next.js adapter
(`@netlify/plugin-nextjs`, OpenNext-based) on every build. It handles SSR,
dynamic routes, the App Router, and the native `/sitemap.xml` and `/robots.txt`
outputs with zero extra configuration.

Configuration lives in [`netlify.toml`](./netlify.toml):

```toml
[build]
  command = "next build"
  publish = ".next"

[[plugins]]
  package = "@netlify/plugin-nextjs"
```

The adapter version is intentionally **not pinned** in `package.json` — Netlify
recommends letting it auto-update so builds always use the latest tested release.

### Environment variables

Set `NEXT_PUBLIC_SITE_URL` in the Netlify site settings to your production URL
(e.g. `https://roomnabs.com` or the generated `*.netlify.app` URL). It is used
for canonical URLs, the sitemap, and `robots.txt`. It defaults to
`http://localhost:3000` for local development.
